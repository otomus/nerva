# API Gateway Examples

REST API gateways that expose Nerva as an HTTP service.

| Language | Framework | Directory |
|----------|-----------|-----------|
| Python | FastAPI | [python/](./python/) |
| TypeScript | Express | [typescript/](./typescript/) |

Both examples demonstrate:
- Auth bridged into Nerva's ExecContext
- Standard POST endpoint for chat
- SSE streaming endpoint
- Health check endpoint

See each subdirectory's README for setup instructions.
